# 🤖 Buddy - Personal AI Assistant

**Buddy** is a customized mobile AI assistant built using **Java** and designed for the Android platform. This project focuses on personalized voice recognition and automation to help manage smartphone functions efficiently.

## 🚀 Features
* **Custom Wake-Word**: Background listening services to detect activation commands.
* **System Automation**: Voice-activated commands to lock screens and open applications.
* **Language Support**: Configured specifically for **Indian English** recognition and speech.
* **GitHub Integration**: Open-source repository management for continuous development.

## 🛠️ Tech Stack
* **Language:** Java
* **Platform:** Android
* **Hosting:** GitHub

## 📁 Project Structure
* `app/src/main/java/` - Contains the Java logic for "Buddy".
* `app/src/main/res/layout/` - Contains the XML UI design files.

## 📝 How to Use
1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the APK on an Android device.
